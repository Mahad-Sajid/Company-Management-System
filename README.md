The Company Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts. The system allows users to manage departments, employees, and projects within a company while maintaining relationships between these entities.

This project demonstrates the implementation of classes, object relationships, dynamic memory management, data organization, and menu-driven interaction in C++.

Features
Department Management
Create new departments
Display department information by ID
View all employees belonging to a department
Remove employees from a department
Employee Management
Create employee records
Display employee details by ID

Assign employees to projects
View all projects assigned to an employee
Find the department of an employee
Project Management
Create projects

Remove employees from projects
View all employees working on a project

System Relationships
Department ↔ Employee
One department can have multiple employees.
An employee belongs to a department.
Employee ↔ Project
An employee can work on multiple projects.
A project can have multiple employees.

This creates a many-to-many relationship between employees and projects.
